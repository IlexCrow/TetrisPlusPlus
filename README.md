Basic Tetris clone because I suffer from boredom.

Limitations: 
 - No audio
 - no T-spin points
 - no levels
 - no point readouts on the screen because text is annoying in SDL and I'm lazy. 

Wins: 
- The tetraminos are quite nice looking.
